24/7 toegang + API

Voor gebruikers en administrators

Geaccrediteerde gebruikers hebben 24/7 toegang tot Accys om pakketjes, bezoeken, rondleidingen of servicewerkzaamheden aan te melden. Realtime is inzichtelijk wie, wanneer en hoelang in uw pand verblijven. Via de API is het (na toestemming) mogelijk om connectie te maken met de beveiligde Accys database om uw eigen software te koppelen aan onze bezoekers- en pakketregistratie.
